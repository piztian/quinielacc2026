Reglas de Puntuacion

  • ⭐ 3 puntos — Resultado exacto (ej: predices 2-1 y es 2-1)
  • 👌 1 punto — Aciertas el ganador o empate (ej: predices 2-1, real 3-0, ambos gana equipo 1)
  • ❌ 0 puntos — No aciertas nada

Como Participar

  • 1. Registrate con tus datos
  • 2. Haz tus predicciones antes de que inicie cada partido
  • 3. Las predicciones se bloquean al inicio de cada partido
  • 4. Las predicciones son definitivas — una vez guardadas no se pueden cambiar
  • 5. El ranking se actualiza conforme terminan los partidos
